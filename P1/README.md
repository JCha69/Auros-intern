# AUROS_인턴 / P1 — 작업 폴더 안내

## 폴더 구조
```
AUROS_인턴/
└── P1/
    ├── KSIA_이슈리포트/
    │   ├── _template_이슈리포트.md   ← 새 보고서 작성 시 참고용 템플릿
    │   ├── [원본 PDF 파일들]
    │   └── [요약 MD 파일들]
    └── KSIA_데일리뉴스/
        └── YYYYMMDD_데일리뉴스.md
```

## 작업 흐름

### KSIA_이슈리포트 (최근 2주 이내 발행분만)
1. (사람) KSIA 사이트에서 최근 2주 이내 보고서 확인
2. (사람) PDF 다운로드 → Claude에게 업로드
3. (Claude) 원본 파일명 그대로 유지 + 동일 이름의 .md 요약 파일 생성
   - 예: `Issue Report No20_반도체 패키징 기술개발 및 산업 동향.pdf`
   - 예: `Issue Report No20_반도체 패키징 기술개발 및 산업 동향.md`
4. 요약은 A4 약 2페이지 분량 + 원본 링크 포함

### KSIA_데일리뉴스 (요청한 날짜 기준)
1. (사람) 해당 날짜의 기사 URL 목록 전달 (가능하면 네이버 미러 대신 원문/발행사 링크)
2. (Claude) 각 기사를 1000자 내외로 요약 + 링크 포함하여 하나의 MD 파일로 생성
   - 예: `20260622_데일리뉴스.md`

## 알아두면 좋은 점
- KSIA 사이트(ksia.or.kr)는 자동 접근이 차단되어 있어, 이슈리포트 PDF는 항상 사람이 직접 다운로드해서 전달해야 함
- 데일리뉴스 중 네이버(n.news.naver.com) 링크는 접근이 차단됨 → 가능하면 KSIA 페이지에서 원문(발행사) 링크를 확인해서 전달하면 누락 없이 처리 가능
